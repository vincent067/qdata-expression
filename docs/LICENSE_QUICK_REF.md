## ✅ 免费使用（AGPL-3.0）

### 允许的场景

| ✅ 可以免费使用 | 说明 |
|----------------|------|
| 个人学习 | 在自己电脑上学习、实验 |
| 研究项目 | 学术研究、毕业设计 |
| 开源项目 | 你的项目也是 AGPL-3.0 开源 |
| 个人 Demo | Portfolio、展示项目 |
| 非营利组织 | 非商业目的使用 |

### 核心要求

如果你修改代码并提供网络服务（包括内部使用），**必须开源你的修改**。

```python
# 示例：开源你的修改

# 2. 修改代码
# 3. 部署为网络服务
# 4. 公开你的修改源码 ← 必须！
```

## ❌ 需要商业许可

### 商业场景

| ❌ 需要购买许可 | 说明 |
|----------------|------|
| 公司内部使用 | 即使不对外收费 |
| 生产环境部署 | 任何商业生产系统 |
| SaaS 服务 | 对外提供付费或免费服务 |
| 商业产品集成 | 集成到你的商业软件中 |
| 云服务托管 | 提供基于本软件的云服务 |
| 超过 5 个用户 | 组织规模超过阈值 |

### 获取商业许可

1. **发邮件**: vincent@qeasy.com
2. **说明场景**: 使用规模、部署方式
3. **获取报价**: 根据实际情况定价
4. **签署协议**: 正式授权文件
5. **开始使用**: 合法合规
